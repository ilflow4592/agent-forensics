# Ecosystem PR Templates

Ready-to-submit content for external project PRs and awesome list registrations.

## How to Use

Each file contains the exact content to submit. Fork the target repo, add the content, and open a PR.

| File | Target | Action |
|------|--------|--------|
| `awesome-lists.md` | awesome-langchain, awesome-llm, awesome-ai-agents | Add entry to list |
| `langchain-community-pr/` | langchain-ai/langchain | Add callback handler |
| `openai-agents-example.py` | openai/openai-agents-python | Add example to cookbook |
| `community-posts.md` | Discord, Forums, Reddit | Copy-paste for community posts |
